# Computational Imaging: From Optics to AI

This repository documents my learning and implementation journey from classical optics to computational imaging and deep learning.

## Topics Covered
- Image formation (Fourier optics)
- Convolution and filtering
- Inverse problems (deconvolution)
- Image denoising
- Deep learning (CNN, U-Net)
- Biomedical imaging applications

## Goal
To build a strong foundation in computational imaging and apply it to real-world problems in biomedical imaging.

## Structure
- Week 1: Image formation
- Week 2: Fourier domain
- Week 3: Deconvolution
- Week 4: Denoising
- Week 5: CNN basics
- Week 6: Segmentation
