## 1. Noise Models

### Salt & Pepper Noise
- Random pixel corruption
- Simulates impulsive noise

### Gaussian Noise
- Additive noise following normal distribution
- Models sensor noise in imaging systems

---

## 2. Filtering Techniques

### Mean Filter
- Linear smoothing filter
- Reduces Gaussian noise but blurs edges

### Median Filter
- Non-linear filter
- Effective for salt & pepper noise

### Bilateral Filter
- Edge-preserving smoothing
- Combines spatial and intensity similarity
