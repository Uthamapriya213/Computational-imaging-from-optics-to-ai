# Computational Imaging Toolkit

This repository contains implementations of classical computational imaging methods:

- Fourier deconvolution
- Wiener filtering
- Coded aperture imaging
- Bayer mosaicing
- Demosaicing
- Compressed sensing reconstruction

## Features

### Deconvolution
- Inverse filtering
- Wiener filtering
- Frequency-domain restoration

### Computational Photography
- Coded apertures
- Bayer sensor simulation
- Demosaicing

### Compressed Sensing
- Random Gaussian sensing matrices
- L1 minimization
- L2 least squares reconstruction
- Error analysis
